## Opis
Autorski pomysł - wykonany w ramach ćwiczeń. Quiz składa się z 20 pytań, z ograniczeniem czasowym (10 sekund na każde pytanie). Za całą interakcję aplikacji odpowiada Java Script. 
Responsywnośc (breakpointy) ustawione na: 576px, 768px, 992px oraz 1200px - mobile first.

## Technologie
JAVA SCRIPT, HTML5 i CSS3
